## Community Profiles API

1. Activate venv
```
python -m venv .venv
source .venv/bin/activate
# Windows .venv\Scripts\activate
```
2. Install requirements `pip install -r requirements.txt`
3. Run app
```
cd src
uvicorn main:app --reload
```
